# InnaCleaning

## Dev Server
- **Port:** 3000 (`npm run dev`)
- **Framework:** Next.js 16.2.1, TypeScript, Tailwind CSS

## What This Is
Cleaning service website for Inna Cleaning with quote estimator, i18n support, and Google OAuth client portal.

## Stack
- Next.js + Tailwind CSS + shadcn/ui
- Supabase (auth + data)
- Resend (email)
- Google Analytics integration

## Session Start
Run `/context` at the start of each session to load the Obsidian vault and check [[InnaCleaning]] for recent work, decisions, and context from past sessions. Use `/save` to log any significant work done during this session.
